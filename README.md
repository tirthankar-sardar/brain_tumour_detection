# 🧠 Brain Tumour Detection using Deep Learning

An AI-powered medical imaging project that detects brain tumours from MRI scans using Deep Learning techniques. This repository demonstrates how Computer Vision and Neural Networks can assist in early diagnosis by classifying MRI images with high accuracy.

## 🚀 Project Overview

Brain tumours are one of the most critical neurological conditions, where early detection can significantly improve treatment outcomes. Manual diagnosis through MRI scans requires expert radiologists and can be time-consuming.

This project leverages **Convolutional Neural Networks (CNNs)** to automatically analyze MRI images and classify whether a tumour is present or not.

> ⚡ Goal: Build an intelligent system that supports faster, more accurate, and scalable brain tumour diagnosis.
